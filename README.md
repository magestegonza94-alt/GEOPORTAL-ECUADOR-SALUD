# GEOPORTAL-ECUADOR-SALUD
Este geoportal permite la visualización y gestión de la accesibilidad a los establecimientos de salud en relación con los poblados, la red vial y el uso del suelo, a nivel de los cantones del Ecuador. El proyecto integra información geoespacial relevante para la planificación y toma de decisiones en salud pública y movilidad territorial.
